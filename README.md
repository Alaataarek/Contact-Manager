# Contact Management CLI App (C#)

A simple **C# Console Application** to manage contacts with:

- Add / Edit / Delete / View / List contacts
- Auto-generated IDs and Creation Date
- Search and Filter contacts
- JSON storage for persistence
- Alphabetical listing of contacts

---

## Requirements

- [.NET 6 SDK](https://dotnet.microsoft.com/download/dotnet/6.0) or higher
- Any code editor (Visual Studio, VS Code, Rider)
